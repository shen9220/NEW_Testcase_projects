## 交互细节

### 上传交互
- 文件拖拽至上传区：显示虚线高亮边框 + "释放以上传" 提示
- 上传成功后自动解析，预览区渲染 Markdown 内容
- 上传失败：红色提示框显示具体错误原因（格式不支持 / 文件过大 / 解析失败）
- 支持重新上传覆盖当前选择

### 生成交互
- 点击「生成测试用例」按钮后：
  1. 按钮变为 loading 状态，显示当前进度步骤（"正在提取功能模块..." → "正在生成模块[登录]用例..." → "正在补充边界用例..." → "正在细化步骤..." → "正在去重..."）
  2. 完成后自动刷新用例表格，显示生成数量
  3. 失败时显示错误信息，日志中记录详细错误原因

### 用例表格交互
- 列宽可拖拽调整，表头固定
- 点击「查看」：右侧滑出抽屉，展示完整步骤列表（action / expected 左右对照）
- 点击「编辑」：弹出模态框，可编辑所有字段，支持步骤的增删改
- 点击「删除」：弹出确认框 "确定删除用例 [标题]？"，确认后逻辑删除
- 批量删除：勾选后顶部出现 "已选 X 条" + "批量删除" 按钮，需二次确认

### 导出交互
- 点击「导出Excel」或「导出XMind」：调用后端 API，自动触发浏览器下载
- 导出中按钮显示 loading
- 导出失败 toast 提示

### Supabase 连接状态
- 🟢 绿色：已连接，数据双重存储
- 🟡 黄色：连接中
- 🔴 红色：未连接，降级为纯本地模式（顶部横幅提示"Supabase 未连接，数据仅保存在本地"）

## 响应式与状态处理

| 状态 | 处理方式 |
|------|----------|
| 加载中 | Skeleton 占位、按钮 loading、进度条 |
| 空数据 | 空状态插图 + 引导文案 "上传 PRD 开始生成测试用例" |
| 网络错误 | Toast 通知 + 表格区域显示错误提示 + 重试按钮 |
| Supabase 离线 | 顶部横幅提示，数据仅本地存储，功能不受影响 |
| 生成超时 | 提示 "生成超时，请重试或减少 PRD 篇幅"，保留已生成部分 |
